# 🗳️ Online Voting System

A secure and user-friendly web-based voting platform that allows users to cast votes online with real-time results. Built to ensure transparency, accessibility, and data integrity.

## ✨ Features

- User Authentication: Secure login/signup for voters and admins
- One Person One Vote: Prevents duplicate voting using user ID verification  
- Real-time Results: Live vote counting with charts and graphs
- Admin Dashboard: Create polls, manage candidates, view analytics
- Mobile Responsive: Vote from any device, anywhere
- Vote Encryption: All votes are encrypted to maintain anonymity
- Candidate Profiles: View candidate details before voting
